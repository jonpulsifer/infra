# CLAUDE.md

Guidance for AI coding agents working in this repository.

## Repository overview

Chezmoi-managed dotfiles: zsh (pure + plugins from `.chezmoiexternal.yaml`), tmux, vim, git, mise global tools, SSH, GPG (Linux), Ghostty, and AI agent configs (Cursor MCP, OpenCode, Claude Code, Gemini).

## Development tools

- **mise**: global CLI versions in `dot_config/mise/config.toml.tmpl`; project tools via per-repo `mise.toml`.
- **chezmoi**: work-specific blocks use `eq .chezmoi.username "jpulsifer"` (see `dot_config/git/config.tmpl`, `.chezmoidata.yaml`).

## Build & validation

```bash
chezmoi diff
chezmoi apply --dry-run
shellcheck dot_local/bin/executable_* .chezmoiscripts/*.sh
```

## Architecture

- **MCP servers**: defined once in `.chezmoidata.yaml`; tool-specific configs generated by `dot_cursor/mcp.json.tmpl`, `dot_config/opencode/opencode.json.tmpl`, and `dot_claude/mcp-servers.json.tmpl`; merged into `~/.claude.json` via `.chezmoiscripts/run_after_configure-ai.sh.tmpl`.
- **Skills**: source under `skills/`; `SKILL.md` wrappers use `include` to deploy to three agent locations.
- **Shell utils**: `dot_local/bin/executable_*` on `PATH` (see `dot_config/zsh/dot_zshrc.tmpl`).

## Git workflow

Commits signed (SSH), Conventional Commits, DCO sign-off. Do not commit directly to `main` — use PRs (`gh pr create`).

## MCP definitions

Single source in `.chezmoidata.yaml`; work-only servers (`moonpay`, `notion`, `linear`) are gated by `eq .chezmoi.username "jpulsifer"` in tool-specific templates.
